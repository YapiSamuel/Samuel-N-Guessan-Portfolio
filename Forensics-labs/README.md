# Digital Forensics Labs

This folder contains hands-on digital forensics investigations focused on analyzing Windows artifacts, reconstructing timelines, and extracting evidence from system files. These labs demonstrate skills in registry analysis, file system examination, and forensic tooling.

## Topics Covered
- Windows Registry analysis (NTUSER.DAT, SAM, SYSTEM)
- Timeline reconstruction and event correlation
- Autopsy and forensic imaging
- Artifact extraction and interpretation
- User activity and program execution analysis
