# SOC Analysis Labs

This folder contains Security Operations Center (SOC) labs focused on alert triage, log analysis, and incident investigation. These exercises demonstrate practical blue-team skills used in real SOC environments.

## Topics Covered
- Alert triage and prioritization
- SIEM investigations (Splunk, Sentinel)
- MITRE ATT&CK mapping
- Log analysis and threat detection
- Incident documentation and escalation
