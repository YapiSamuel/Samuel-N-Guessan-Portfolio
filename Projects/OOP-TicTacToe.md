# 🎮 Object‑Oriented Tic‑Tac‑Toe (Dynamic Board + AI)

This project is an advanced Tic‑Tac‑Toe game built using Object‑Oriented Programming principles.  
The game supports **custom board sizes**, **human vs human**, and **human vs AI** gameplay.

## 🧠 Key Features
- Dynamic board size (3x3, 4x4, 5x5, etc.)  
- Play against another human or an AI opponent  
- AI decision‑making using programmed logic  
- Fully object‑oriented design (classes for Board, Player, Game, AI)  
- Input validation and clean game‑loop structure  

## 🧰 Skills Demonstrated
- Object‑Oriented Programming (encapsulation, inheritance, polymorphism)  
- Game logic design  
- AI decision‑making  
- Clean code architecture  
- User input handling  

## 🎯 Outcome
A polished, flexible, and replayable game that showcases strong OOP fundamentals and the ability to build interactive applications with AI components.
