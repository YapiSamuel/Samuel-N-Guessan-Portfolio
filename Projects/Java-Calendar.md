# 📅 Java Calendar Application

This project is a Java‑based calendar tool designed to help users view dates, navigate months, and interact with a simple scheduling interface.

## 🔍 Features
- Displays monthly calendar layout  
- Navigate forward and backward through months  
- Highlights current date  
- Clean and simple terminal‑based UI  
- Built entirely in Java  

## 🧰 Skills Demonstrated
- Java fundamentals  
- Date and time manipulation  
- Control flow and data structures  
- User interface logic in console applications  

## 🎯 Outcome
A functional and user‑friendly calendar program demonstrating strong Java programming skills and the ability to build practical utility applications.
