# 🤝 Peer Programming Project

This project was built as part of a collaborative peer‑programming assignment. The goal was to practice real‑world teamwork, version control, and shared problem‑solving using Git and GitHub.

## 🔍 What We Built
- A jointly developed program created through pair‑programming workflows  
- Shared responsibilities across design, implementation, and debugging  
- Frequent commits, pull requests, and code reviews  
- Emphasis on communication, collaboration, and clean coding practices

## 🧰 Skills Demonstrated
- Git branching, merging, and conflict resolution  
- Collaborative development using GitHub  
- Writing maintainable and readable code  
- Team‑based problem solving  

## 🎯 Outcome
This project strengthened my ability to work effectively in a development team, communicate technical ideas, and contribute to a shared codebase — all essential skills for real‑world engineering environments.
